### 10.1 获取站点的当日生产情况
#### GET /stations/production

#### 请求参数
|参数名|参数类型|是否必填|含义|
|:---:|:---:|:---:|:---:|
|name|String|否|要获取的站点的名称，目前可取值为CPF1、DS2、DS1、MJ-E20、MJ-E22、MJ-E24、MJ-F25|

#### 请求示例
```json
{
    "name": "DS2"
}
```

#### 返回参数

|data|Object|站点的生产信息|

### 10.2 获取站点过去数天内的生产情况
#### GET /stations/production/history

#### 请求参数
|参数名|参数类型|是否必填|含义|
|:---:|:---:|:---:|:---:|
|name|String|否|要获取的站点的名称，目前可取值为CPF1、DS2、DS1、MJ-E20、MJ-E22、MJ-E24、MJ-F25|
|count|Integer|否|需要几天的数据，不提供则获取7天的|

#### 请求示例
```json
{
    "name": "DS2"
}
```

#### 返回参数
|参数名|参数类型|含义|
|:---:|:---:|:---:|
|data|Array|站点的生产信息|
|data[i].date|DateString|日期|
